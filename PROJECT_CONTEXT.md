# VIROXA TRACKER

## Vision

Viroxa adalah custom affiliate tracker yang terinspirasi dari BeMob, tetapi menggunakan stack sendiri:

* Next.js
* Supabase
* Cloudflare Worker
* Vercel

Tujuan:

* Tracking clicks
* Tracking conversions
* Campaign management
* Analytics dashboard
* Realtime reporting

Bukan sekadar shortlink analytics.

---

## Current Status

### Worker

Sudah berjalan.

Features:

* Redirect engine
* Click tracking
* Conversion tracking
* Postback endpoint

Flow:

Visitor
→ /go/:campaign
→ generate clickid
→ save click
→ redirect offer

Postback
→ /postback
→ save conversion

## Campaign Concept

Target concept similar to BeMob.

User creates campaign.

Example:

Campaign Name:
IMO Main

Destination URL:
https://slow.gen-kranjang.site/...

Tracking URL generated by Viroxa:

/go/imo-main

Visitor:

/go/imo-main
→ tracking
→ redirect offer

All reporting comes from Viroxa tracking layer.
